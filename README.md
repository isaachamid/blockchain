- Decentralized Blockchain

##

## How to run

### 1. Clone repository:

```bash
git clone git@github.com:isaachamid/blockchain.git
cd blockchain
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run Instance 1

```bash
npm run node_1
```

### 2. Test in your browser

```bash
http://localhost:3001/blockchain
```
